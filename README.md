## 🚀 安装步骤

### 方式一：一键安装（推荐）

```
1. 解压七虾技能包v3.1.1.zip 到任意目录
2. 右键 → 一键安装脚本.ps1 → 使用 PowerShell 运行
3. 等待提示"安装完成"
4. 重启 OpenClaw
```

### 方式二：手动安装

```
1. 复制 skills/ 文件夹到以下两个位置：
   ① C:\Users\你的用户名\.openclaw\skills\
   ② C:\Users\你的用户名\.openclaw\workspace\.agents\skills\

2. 复制 外部依赖/ 文件夹到：
   F:\My_WenAn\OpenClaw-File\七虾技能包v3.1.1\外部依赖\
   （如无F盘，改为你的数据盘，路径需与SKILL.md中的引用一致）

3. 重启 OpenClaw
```

## 🎯 使用指南

### 全自动模式（一句话搞定）
```

```
编排器自动执行：选题→框架→大纲→文案→标题→图片→审核，全程无需干预。

## ❓ 常见问题

### Q1：安装后不生效？
确保技能文件在两个位置都已就绪：
- `%USERPROFILE%\.openclaw\skills\*xia\`
- `%USERPROFILE%\.openclaw\workspace\.agents\skills\*xia\`

### Q2：外部依赖放哪？
默认路径：`F:\My_WenAn\OpenClaw-File\七虾技能包v3.1.1\外部依赖\`
如无F盘，放数据盘后需修改文案虾/图片虾的SKILL.md中引用路径。

### Q3：如何更新技能包？
1. 下载新版ZIP
2. 重新运行一键安装脚本（会覆盖旧版）
3. 重启OpenClaw

### Q4：文案虾参考的朋友圈指南在哪看？
解压后 `外部依赖/朋友圈运营完全指南.md`，含文案三板斧、配图尺寸、发布时段、互动钩子等完整规则。

### Q5：全自动模式和手动分步有什么区别？
- **全自动**：一句话启动，全程自动完成，适合批量生产
- **手动分步**：每环节确认后再推进，适合精打细磨
