## 第一章：七虾技能包是什么？

### 1.1 一句话解释

**七虾技能包是一个"AI写作流水线工厂"**。

你只需要告诉它"帮我写一篇关于XX的文章"，它就会自动帮你完成从选题到审核的全部工作，最后给你一篇可以直接发布的好文章。

就像你去饭店点餐——你只说"来一份宫保鸡丁"，厨房里摘菜、切肉、炒菜、装盘一条龙自动完成。**七虾技能包就是你的写作厨房**。

### 1.2 它能帮你做什么？

- 🎯 不会写选题？→ 它帮你找热门话题
- 📝 不知道怎么搭文章架子？→ 它帮你设计大纲
- ✍️ 写文章费劲？→ 它帮你写初稿并去掉AI味
- 📊 不会给文章起好标题？→ 它一口气给你5个标题备选
- 🖼️ 不知道配什么图？→ 它给你写图片提示词（中英文都有）
- ✅ 写完不敢发？→ 它帮你全面检查质量

### 1.3 谁在用七虾技能包？

- **公众号博主**：一周3篇文章 → 半小时搞定
- **小红书博主**：日更内容 → 10分钟出一篇
- **知识星球主理人**：每天输出干货 → 不再灵感枯竭
- **朋友圈营业**：每天3条高质量文案 → 自动生成
- **企业新媒体运营**：多平台矩阵内容 → 一次搞定

### 1.4 需要什么条件和基础？

| 条件 | 说明 |
|:-----|:------|
| 电脑操作 | 会打字、会点击鼠标即可 |
| 技术背景 | **完全不需要**——不懂代码、不懂AI也能用 |
| 安装软件 | 只需安装一个叫"OpenClaw"的工具（见第二章）
| 费用 | 模型调用有少量费用，写一篇约几分钱到几毛钱 |
| 英文 | 不需要，全程中文操作 |

## 第二章：准备工作——安装与启动

### 2.1 什么是OpenClaw？

**OpenClaw是一个AI助手平台**，七虾技能包是装在上面的一套写作工具。你可以把OpenClaw理解成手机（操作系统），七虾技能包就是装在上面的一个App（比如微信）。

### 2.2 检查七虾技能包是否已安装

**第一步**：打开OpenClaw的界面（通常在浏览器打开一个页面）。

**第二步**：在对话框里输入以下任意一句话试试：

```
帮我写一篇关于AI的文章
```

或者：

```
七虾全流程，写一篇办公效率的文章
```

**第三步**：观察AI的回复。

- ✅ **如果它开始问你写什么方向、然后自动给你出选题** → 恭喜你，七虾技能包已经装好了，可以跳到**第四至五章**开始使用！
- ❌ **如果它说"我不清楚七虾是什么"或随便写了段话** → 说明还没安装，请继续看下面的安装教程。

### 2.3 如果还没安装——安装教程

> ⚠️ **注意**：以下步骤如果看不懂，可以找你帮你装OpenClaw的人帮忙。

**步骤1：确保OpenClaw已安装并运行**

OpenClaw是七虾技能包的"运行底座"，需要先装好。如果你已经有了OpenClaw的界面，跳过此步。

**步骤2：下载七虾技能包**

从百度网盘下载最新的七虾技能包压缩包（找提供给你技能包的人要链接），解压到电脑上。

**步骤3：安装技能包**

在OpenClaw界面中操作：
1. 找到「技能管理」或「安装技能」的入口
2. 选择「从文件夹安装」
3. 选中解压后的七虾技能包文件夹
4. 点击安装

安装成功后，你会看到技能列表中出现七虾相关的技能（选题虾、框架虾等七八个名称）。

**步骤4：验证安装**

输入上面测试的那句话，看它是否开始自动走流程。

> 💡 **如果卡住了怎么办？** 直接问帮你装OpenClaw的人，或者跳到[第十二章：故障排除](#第十二章故障排除)。

## 第三章：认识七只"虾"——功能速览

七虾技能包里有**7只分工不同的"虾"**（"虾"就是AI助手的意思，每一只负责一个环节），加上一个总调度员（编排器）。

### 3.1 一张图看懂全流程

```
你 说"帮我写篇文章"
        │
        ▼
┌──────────────────────────────────────────────────────────┐
│                  总调度员（编排器）                          │
│     判断是自动跑还是手工跑，依次调遣七只虾                     │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  ┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐      │
│  │①选题虾 │──▶│②框架虾 │──▶│③大纲虾 │──▶│④文案虾 │      │
│  │  找话题  │   │ 选结构  │   │ 搭骨架  │   │ 写正文  │      │
│  └────────┘   └────────┘   └────────┘   └────────┘      │
│       │            │           │           │             │
│       ▼            ▼           ▼           ▼             │
│  ┌────────┐   ┌────────┐   ┌────────┐                   │
│  │⑤标题虾 │──▶│⑥图片虾 │──▶│⑦审核虾 │──▶ 出稿！🎉     │
│  │  起标题  │   │ 配图方案  │   │ 检查质量  │               │
│  └────────┘   └────────┘   └────────┘                   │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 3.2 每只虾具体做什么？

| 编号 | 名称 | 外号 | 它干什么 | 大约耗时 |
|:----:|:----|:----|:---------|:--------:|
| ① | **选题虾** | 灵感雷达 | 搜热点、找角度，给你3-5个可写的选题 | 30秒 |
| ② | **框架虾** | 建筑设计师 | 推荐最合适的文章结构（比如"先戳痛点再给解药"这种写法） | 15秒 |
| ③ | **大纲虾** | 钢铁骨架 | 搭出文章的详细大纲（第一章写什么、第二章写什么） | 30秒 |
| ④ | **文案虾** | 写作机器人 | 按照大纲写出完整的文章，还去掉了AI味 | 60-90秒 |
| ⑤ | **标题虾** | 标题魔术师 | 为每个平台起5个好标题，给出选择理由 | 20秒 |
| ⑥ | **图片虾** | 视觉设计师 | 写出封面图和文中配图的提示词（给AI绘图工具用） | 40秒 |
| ⑦ | **审核虾** | 质量检查员 | 从10个维度检查文章质量，合格才能发布 | 30秒 |

### 3.3 什么是"写作框架"？

你可能会在操作过程中听到框架虾提到一些英文缩写，这里简单解释一下：

| 框架 | 中文意思 | 打个比方 |
|:-----|:---------|:---------|
| **P&P** | 痛苦→快乐 | 先说到你的痛处，再给你解药。比如："你是不是加班到崩溃？其实有个工具能帮你省3小时" |
| **PAS** | 问题→焦虑→方案 | 先抛出问题，再告诉你有多严重，最后给方案。比如："你还在一行一行做工资条？小心出错被扣工资！试试这个自动工具" |
| **PPP** | 画面→承诺→行动 | 先描绘美好画面，再承诺教给你，最后教步骤。比如："想象一下10分钟做完一天的工作……这篇文章就教你这么做" |
| **故事框架** | 起→承→转→合 | 像讲故事一样，有开头、冲突、转折、结尾。比如："三年前我还是个小白，直到有一天……" |
| **清单框架** | 并列罗列 | 1、2、3、4一条条列出来。比如："10个让你效率翻倍的AI工具，第7个绝了" |

> 💡 **新手不用记这些框架名字**——全自动模式下，系统会自动帮你选择最适合的。你只需要享受结果就行。

## 第四章：两种工作模式

七虾技能包有**两种使用模式**，你可以根据自己的需要选择：

### 4.1 全自动模式：一句话出稿（⭐ 新手推荐）

**适合谁**：想快点出文章的、不想了解太多细节的、刚上手的新手  
**怎么做**：直接说"写一篇关于XXX的文章"  
**特点**：从选题到审核全部自动完成，你只需要：
1. 说一句"我要写什么"
2. 中间如果有大纲生成，需要确认一下（大约3-5秒）
3. 等待3-5分钟
4. 拿到完整文章

### 4.2 手动模式：一步一步来（进阶）

**适合谁**：对文章质量要求很高、想参与每个环节的、有特定写作需求  
**怎么做**：说"手动创作一篇关于XXX的文章"  
**特点**：每个环节完成后都会停下来问你"这样可以吗？"，你确认后才继续下一步：
1. 选题虾出选题 → **你看完后说"用第2个"**
2. 框架虾推荐框架 → **你说"用P&P框架"**
3. 大纲虾出大纲 → **你看完后说"没问题"**
4. 文案虾写正文 → **你看完后说"可以了"**
5. 标题虾出标题 → **你选1个**
6. 图片虾出提示词 → **你选风格**
7. 审核虾出报告 → **你说"通过"**

> ⚠️ **新手第一次用，建议先用全自动模式感受一下流程！**

## 第五章：全自动模式——一句话出稿（推荐新手）

### 5.1 准备工作

开始之前，想清楚这几个问题（不需要很精确，大概就行）：

- ☐ **写什么话题？**（比如"AI对职场的影响""Excel做工资条""小红书怎么涨粉"）
- ☐ **发在哪个平台？**（公众号/小红书/知乎/今日头条/朋友圈，不指定的话默认公众号）
- ☐ **大概写多少字？**（短文800字/中篇1500字/长文2500字）

### 5.2 开始操作——分步步骤

**第1步：告诉它你要写文章**

在对话框里输入（任选一种）：

```
帮我写一篇关于AI对职场影响的文章
```

或者更具体一点：

```
写一篇关于DeepSeek自动做工资条的文章，发公众号，1500字左右
```

**第2步：系统自动走流程**

你什么都不用做，看着它自动运行。你会看到类似这样的信息输出：

```
🦐 七虾全流程启动！

▶ 阶段1：选题虾 —— 正在挖掘热门选题...
▶ 阶段2：框架虾 —— 自动选择写作框架...
▶ 阶段3：大纲虾 —— 正在生成文章大纲...
```

**第3步：确认大纲（唯一需要你做的一步）**

当系统生成大纲后，会问你类似这样的话：

```
📝 文章大纲已生成

### 开头
描绘财务人员月末做工资条的痛苦场景（★金句位置）

### 第一章：你用Excel的方式可能全错了
- 讲很多人还在用错误方式做工资条
- 金句：...

### 第二章：DeepSeek的真正价值
- 介绍DeepSeek在Excel中的实际应用

### 第三章：手把手教你自动生成工资条
- 具体的操作步骤

### 结尾
总结要点，鼓励实践

---

> ⚠️ 回复「确认大纲」继续，或告诉我需要修改的地方
```

这时你需要：
- ✅ **觉得大纲没问题** → 回复：**确认大纲**（或者"可以""OK""没问题"）
- ✏️ **想修改** → 直接说修改意见，比如"第一章改成讲手动做工资条的错误"

**第4步：等待系统自动完成剩下的工作**

你确认大纲后，系统会自动：
1. 文案虾写完整文章（约60-90秒）
2. 标题虾起标题（约20秒）
3. 图片虾写配图提示词（约40秒）
4. 审核虾检查质量（约30秒）

**第5步：收到最终结果 🎉**

你会收到类似这样的结果：

```
🎉 七虾全流程——闭环完成！

✅ 文章已交付
✅ 标题方案已交付
✅ 封面图+配图方案已交付
✅ 审核通过，可以发布

📁 文件已保存到：F:\...\自媒体文章\文章标题\

需要我帮你发布到哪个平台吗？
```

## 第六章：手动模式——每步确认写作（进阶）

### 6.1 什么时候用手动模式？

- 你有特别明确的写作方向，不希望AI自由发挥
- 你参与感很强，想亲自决定每一步
- 你第一次用，想看看每个环节都输出什么

### 6.2 开始操作——分步步骤

**第1步：告诉它你要手动创作**

在对话框里输入：

```
手动创作一篇关于AI对职场影响的文章
```

或者：

```
手动写一篇公众号文章，关于Excel做工资条
```

**第2步：选题虾出选题，你来选**

```
🎯 选题推荐（共3个）

选题1：AI不会取代你，但会用AI的人会 ⭐⭐⭐⭐⭐
选题2：prompt工程师年薪50万，普通人如何入门
选题3：我用DeepSeek做了一个自动工资条系统

⏸️ 请回复：
- 直接输入选题编号（如"选题1"）确认
- 或告诉我你的其他想法
```

✅ 你回复：**选题2**（或其他你想要的）

**第3步：框架虾推荐框架，你来选**

```
🎯 推荐框架

推荐：P&P框架（先戳痛点再给解药）
理由：适合情感共鸣+实用干货
备选：PPP框架（描绘画面+分步教学）

想试试小白测试吗？回复"帮我选"可做3道选择题快速推荐。

⏸️ 请选择框架名称
```

✅ 你回复：**P&P框架**（或你想要的）

**第4步：大纲虾出大纲，你确认**

展示大纲后，系统会暂停等你回复。
✅ 你回复：**确认大纲**（或修改意见）

**第5步：文案虾写正文，你确认**

展示完整文章后，系统会暂停等你回复。
✅ 你回复：**可以了**（或修改意见）

**第6步：标题虾出标题，你选**

```
🎯 公众号标题推荐

标题1：工资条还在一行一行复制？DeepSeek全自动生成
标题2：做了10年财务才明白：工具选对，效率翻倍
标题3：深夜还在做工资条？这个AI工具救了我
...

⏸️ 请选择标题（输入"标题1/2/3"）
```

✅ 你回复：**标题1**

**第7步：图片虾选风格，你确认**

```
🎨 7种视觉风格

1. 简约现代风（推荐）—— 干净大方，适合科技效率类
2. 科技未来风 —— 炫酷光效，适合AI主题
3. 温暖治愈风 —— 柔和色调，适合情感类
...

推荐：简约现代风

⏸️ 请输入风格编号或名称确认
```

✅ 你回复：**简约现代风**（或直接说"确认"）

图片虾生成提示词后，再次展示给你确认。
✅ 你回复：**确认**

**第8步：审核虾出报告，你决定通过或修改**

```
📊 十维度审核报告

P0问题：0个 ✅
P1问题：0个 ✅
CMO合规：✅ 优秀
SEO合规：✅ 优秀

结论：审核通过 ✅

⏸️ 请回复"通过"或说明要修改的部分
```

✅ 你回复：**通过** 🎉

### 6.3 手动模式流程图

```
你 说"手动创作一篇关于XX的文章"
    │
    ▼

## 第八章：图片提示词——给AI画图师看的话

### 8.1 什么是"提示词"？

"提示词"就是你给AI画画工具说的话，告诉它"帮我画一张什么样的图"。**图片虾就是帮你写这段话的**。

### 8.2 图片虾不生成图片

请特别注意：**图片虾只写文字描述，不直接生成图片**。你需要把提示词复制到以下任意一个AI绘图工具里使用：

| 通义万相 | tongyi.aliyun.com | 阿里出品，免费 |
| 文心一格 | yige.baidu.com | 百度出品，免费 |

### 8.3 占位符是什么？

在文章正文中，图片虾会插入类似这样的标记：

```
**【MEDIA:封面_公众号版.png】**
```

这叫做"占位符"，意思是"这里应该放一张图片"。你可以稍后把生成的图片替换进去。

**占位符命名规则：**
- 封面图：`封面_公众号版.png`、`封面_小红书版.png`
- 文中插图：`插图1_主题_公众号版.png`、`插图2_主题_公众号版.png`

## 第十一章：新手常见问题（FAQ）

### Q1：写一篇文章要多久？

**全自动模式**：约3-5分钟。  
**手动模式**：约5-10分钟（取决于你确认的速度）。  
**比你自己写快多少**：你自己写一篇文章至少1-2小时，七虾帮你省了95%的时间。

### Q2：写出来的文章质量怎么样？

文章经过**四重保障**：
1. **文案虾**写初稿并去AI味
2. **标题虾**起好标题
3. **图片虾**配好图
4. **审核虾**从10个维度全面检查

审核通过的文章，质量已经达到可以直接发布的标准。

### Q3：完全不符合我的风格怎么办？

两个办法：
1. **手动模式**：每个环节你都可以修改，按你的要求来
2. **告诉它修改**：直接说"第一章改一下，我想加入一个案例"，它会立即修改

### Q4：可以一个选题写多篇文章吗？

当然可以！每次说"再写一篇关于XX的文章"，它会自动生成内容不同的文章。标题虾还支持"首发/二发"标题策略。

### Q5：我想发到多个平台怎么办？

告诉它目标平台即可，比如：
```
写一篇关于AI办公的文章，发到公众号、小红书、朋友圈三个平台
```
系统会一次产出三个版本的稿件，分别适配不同平台。

### Q6：一定要联网才能用吗？

**是的**。七虾技能包需要连接AI模型服务，所以需要联网。正常的4G/5G/WiFi都可以。

### Q7：配图真的不直接生成吗？

**对，图片虾只写提示词（提示词）**。你需要把提示词复制到AI绘图工具（比如Midjourney、即梦AI、通义万相）里生成图片。

### Q8：我不懂英文，图片虾写的英文提示词怎么办？

**完全不用担心**！图片虾产出的提示词**同时包含中文和英文两个版本**：
- 你用**中文版**给国内AI绘图工具用（即梦AI、文心一格、通义万相）
- **英文版**给国外工具用（Midjourney、DALL-E）

### Q9：朋友圈文案为什么要分时段发？

不同时间发朋友圈，效果天差地别：

| 时段 | 最佳内容 | 原因 |
|:-----|:---------|:-----|
| 早7-9点 | 干货技巧 | 通勤刷圈高峰 |
| 午12-14点 | 轻松生活、知识 | 午休刷手机 |
| 晚22-23点 | 故事、情感共鸣 | 睡前高频刷圈 |

图片虾会在每条文案里标注**建议时段**，你按时间发就行。

### Q10：出错了怎么办？

先别慌，看[第十二章：故障排除](#第十二章故障排除)。如果还解决不了，告诉帮你装系统的人，把错误信息复制给他看就行。

### Q11：大纲确认后还能改吗？

可以。如果文章写完后发现点问题，直接说"第三章改一下，我想……"，文案虾会修改。修改完成后，后面的环节会自动调整。

## 第十二章：故障排除

### 12.1 常见问题与解决办法

| 问题 | 可能原因 | 解决方法 |
|:-----|:---------|:---------|
| 输入后没反应 | 网络不稳定或模型忙 | 耐心等待30秒，或把话重新说一遍 |
| 中途中断了 | 网络波动 | 再说一遍"继续刚才的写作" |
| 文章有AI味 | 润色没到位 | 说"帮我去一下AI味再写一遍" |
| 标题不满意 | 自动选的不合你意 | 说"换一个标题" |
| 图片提示词看不懂 | 术语太专业 | 直接说"用中文写，简单点" |
| 提示说文件保存失败 | 文件夹不存在 | 说"改成保存到桌面" |
| 审核不通过 | 质量有瑕疵 | 系统会自动回退修改，你也可以说"帮我改掉这些问题" |
| 忘记刚才写到哪里了 | 中途离开太久 | 说"继续，从大纲环节开始" |

### 12.2 手动模式卡住了怎么办？

手动模式下，每个环节完成后会停住等你确认。如果卡住了：
1. 看看系统是否在等你回复（屏幕最下方通常有"⏸️ 等待确认"提示）
2. 回复"继续"或"确认"即可
3. 如果想跳过当前环节，说"跳过"

### 12.3 写了不满意，怎么重来？

**只想改一部分：**
```
第三章的案例换一个，改成XX行业的
```

**完全重写：**
```
重新写一遍，这次用故事框架
```

**换平台：**
```
改成小红书版，短一点
```

### 12.4 遇到英文报错怎么办？

如果你在对话框里看到一段看不懂的英文，不要慌：
1. **截图**发给帮你安装的人
2. 或者**复制那段文字**发过去
3. 大概率是网络小波动，再说一遍就行

### 12.5 多个"虾"之间如何概念区分？

| 你听到的 | 大白话解释 |
|:---------|:-----------|
| "选题虾" | 专门帮你想写什么的AI |
| "框架虾" | 专门帮你定文章结构的AI |
| "大纲虾" | 专门帮你搭文章骨架的AI |
| "文案虾" | 专门帮你写正文的AI |
| "标题虾" | 专门帮你起标题的AI |
| "图片虾" | 专门帮你写配图描述的AI |
| "审核虾" | 专门帮你检查质量的AI |
| "编排器" | 管理上面7只虾的总管 |
